Command Info

Prefix: /
{} - Required Option
() - Optional Option

Utility Commands

getAvatar is used to get the avatar of a selected user

Usage: /getAvatar {user}
Permissions: everyone

Get an embed that you can select the category you want to get the commands from!

Usage: /help
Permissions: everyone

Get the info of the server you are in! Get server member count, role count, and more!

Usage: /serverInfo
Permissions: everyone

Get the info of the user you selected or yourself! Get banner link, avatar link, role count, roles and more!

Usage: /userInfo (user)
Permissions: everyone

Moderation Commands

You can use addRole to add roles to a user, the user must be a lower rank than you. You cannot add roles to members above you!

Usage: /addRole {role} {user}
Permissions: moderators

Just like the addRole command but removes the role from the user, same rules apply.

Usage: /removeRole {role} {user}
Permissions: moderators

Allows you to send an embed in a selected channel, you can also select if you want to mention @everyone!

Usage: /announce {channel} {title} {description} {mention} {color}
Permissions: moderators

Used to display the logs from a user.

Usage: /logs {warns/notes} {user}
Permissions: moderators

All moderation commands compressed into one! You can ban, kick, and timeout users using this single command! You can also set modlogs too to track the usage of this command.

Usage: /mod {ban/kick/timeout} {user} (time) (reason)
Permissions: moderators

Used to setup, delete, or replace the modlogs channel!

Usage: /modlogs {setup/delete/replace_channel} {channel}
Permissions: moderators

Mute a user by giving them a role temporarily until the time you selected.

Usage: /mute {user} {time} (reason)
Permissions: moderators

add a note to a user, the note can be later accessed using the /logs command!

Usage #1: /logs add {user} {note}
Usage #2: /logs edit {id} {new_note}
Usage #3: /logs remove {id}
Permissions: moderators

warn a user by adding a message to their logs and sending them a dm, the logs can be later accessed by doing /logs!

Usage #1: /warn add {user} (reason)
Usage #2: /warn remove {id}
Permissions: moderators

add a slowmode to the channel you are in.

Usage: /slowmode {duration} Permissions: moderators

Systems

This system creates a ticket system that is customizable which your users can get support by your support team. This command can greatly improve productivity in your server!

Setup: /tickets setup {channel} {category} {support-role} {ticket-logs}
Delete: /tickets delete
Permissions: admins

This system creates a suggestion system that could help you find the perfect addition by your community!

Setup: /suggestions setup {channel} -- admins
Suggest: /suggestions suggest {suggestion} -- everyone

This system sets up the welcome and leave events allowing you to better track who entered and who left your server!

Welcome Events

Setup: /events welcome {channel} {enable} {chat} {rules} {mention-user} (welcome-image)

Leave Events

Setup: /events leave {channel} {enable}
Permissions: admins
